Signup Form  (HTML & CSS Only)
Description

This is a modern and responsive signup form built using only HTML and CSS. The project demonstrates how to create a clean, user-friendly, and visually appealing signup interface without using any JavaScript.

The form includes:

Full Name, Email, Password, and Confirm Password fields

Centered submit button for a polished layout

Hover effect on the form card for interactive design

Background image to enhance visual appeal

This project is ideal for educational purposes, portfolio demonstration, or as a starting point for integrating backend functionality later.

Key Features

Responsive Design – Works well on desktop and mobile screens

Interactive Form Card – Hover effect creates a modern, lifted appearance

Centered Submit Button – Easy to locate and visually balanced

Clean UI – Minimalistic design using gradients, rounded corners, and shadows

Background Image – Adds depth and professional look to the page

Technologies Used

HTML5 – Structure and form elements

CSS3 – Styling, layout, hover effects, background image, and responsiveness
